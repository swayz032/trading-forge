## Agent 1: Strategy Files (19 files scanned)

### Confirmed Bugs — FIXED

**BUG P2: ict_swing.py:214-220 — Same-bar entry+exit**
- Exit logic fires on the entry bar. When `entry_long[i]=True`, `in_long` is set True, then the exit check at line 214 (`if in_long:`) runs on the same bar. If BSL target already exceeded or bearish BOS on the entry bar, both `entry_long[i]` and `exit_long[i]` are True. vectorbt silently drops the entry.
- **Fix:** Changed `if in_long:` to `if in_long and not entry_long[i]:` (and same for shorts).

**BUG P2: propulsion.py:165-168 — Same-bar entry+exit (long)**
- Entry sets `long_active=True` and `long_mean_thresh`, then exit check on same bar fires if `closes[i] < long_mean_thresh`. Zone bottom can be below mean threshold, so a valid entry that closes between them triggers both signals.
- **Fix:** Restructured to check exits BEFORE entries (standard pattern in codebase).

**BUG P2: propulsion.py:170-173 — Same-bar entry+exit (short)**
- Same structural issue on the short side.
- **Fix:** Same restructure — exits before entries.

**BUG P3: propulsion.py:136-163 — Simultaneous long AND short entry**
- Independent `if not long_active` and `if not short_active` checks allow both to fire on the same bar if zones overlap.
- **Fix:** Added `and not short_active` / `and not long_active` guards.

## Agent 2: Indicator Files (9 files scanned, 1994 lines)

### Confirmed Bug — FIXED

**BUG P2: order_flow.py:detect_bullish_ob/detect_bearish_ob — Shifted swing indices**
- `detect_swings` shifts swing indices forward by `+lookback` for confirmation delay (market_structure.py:44). These shifted indices are passed to `_find_bullish_obs`/`_find_bearish_obs`, which search backward from the shifted position. With lookback=5, this means the search includes 4 bars AFTER the actual swing point — finding order blocks at wrong candles.
- **Impact:** Every OB-based strategy (ict_swing, breaker, mitigation, unicorn) identifies OBs from candles that formed AFTER the actual swing reversal instead of BEFORE it. Entry zones are wrong.
- **Fix:** Added `lookback` parameter to `detect_bullish_ob` and `detect_bearish_ob`. When >0, subtracts the shift from swing indices before searching. Updated all 4 strategy callers to pass their lookback value.
